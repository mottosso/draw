### An intuitive collaborative drawing web based tool

Collaborative real-time drawing, sketching & painting

### Usage

With Docker

```bash
git clone https://github.com/JohnMcLear/draw.git
cd draw
. build.sh
. run.sh
```

Now browse to http://localhost:3002!
